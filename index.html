RAF Banka 3 API (0.7.0)

Download OpenAPI specification:

URL: https://github.com/RAF-SI-2025/Banka-3-Backend

HTTP API exposed by the gateway service. Most endpoints require an Authorization: Bearer $ACCESS_TOKEN header. A subset of write endpoints additionally require a TOTP: <6-digit code> header (noted per-endpoint).

All paths are mounted under /api, except for /healthz which is exposed at the root.

Permission model — middleware accepts either a permission string (e.g. manage_clients) or a role check (role:client, role:employee, role:client|employee). The synthetic admin permission bypasses every secured(...) check, so admin tokens implicitly have access to every route. The synthetic supervisor permission gates the actuary supervisor portal, the supervisor orders/tax portals, and the exchange open-override toggle.

Money values — balances, limits, amounts, prices, strike prices, premiums, commissions and tax debts are integer minor units (e.g. RSD para, EUR/USD/CHF cents) and are typed as integer (int64). Rates and percentages (exchange rate, dividend yield, implied volatility) are number (double).

Timestamps are seconds since epoch (UTC) returned as integer (int64) unless explicitly typed as a date-time string (RFC3339).
